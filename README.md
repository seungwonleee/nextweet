## 프로젝트

Next.js를 사용하여 SSR이 가능한 트위터 클론 프로젝트

## 기능

1. 회원가입, 로그인, Google 소셜 로그인
2. 게시글, 댓글 작성, 수정, 삭제
3. 해쉬태그 검색기능
4. 게시글 리트윗
5. 인피니티 스크롤

## 사용 기술

### Front-end

> Next.js, ReactJS, Redux, Redux-Saga, styled-components, Ant-Design

### Back-end

> Node.js, ExpressJS, Nginx, Sequelize

### Database

> MySQL

### Deploy

> AWS (EC2, S3, Route 53, Lamda)
